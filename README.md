# MK Booking Demo

Moderní rezervační systém vytvořený v Next.js jako demo projektu pro portfolio.

## Funkce

- výběr služby
- výběr specialisty
- výběr data a času
- formulář rezervace
- souhrn rezervace
- moderní UI inspirované SaaS aplikacemi

## Technologie

- Next.js
- React
- Tailwind CSS
- Lucide Icons

## Demo

Projekt simuluje profesionální booking flow pro služby a konzultace.
